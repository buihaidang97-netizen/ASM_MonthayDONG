Project Description:
This project presents a Power BI dashboard for Samsung Electronics. The dashboard uses sales, product, region, inventory, production and supplier performance data to support business decision-making. It helps analyse sales performance, inventory levels, production planning and supplier performance.

Files Included:
1. Power BI For Everyone Template(2).pbix
   - This is the Power BI dashboard file.

2. Samsung_SupplyChain_PowerBI_FULL(4).xlsx
   - This is the Excel dataset used for the dashboard.

How to Open the Project:
1. Open Microsoft Power BI Desktop.
2. Open the PBIX file.
3. If Power BI asks for the data source, reconnect it to the Excel file included in this folder.
4. Refresh the data to load the latest tables and visuals.

Main Dashboard Pages:
1. Overview Page
   - Shows key performance indicators and overall business performance.

2. Sales Analysis Page
   - Shows sales by product, category, region and time.

3. Inventory and Supplier Performance Page
   - Shows inventory level, stock status, supplier delivery performance and defect rate.

Purpose of the Dashboard:
The purpose of this dashboard is to transform raw business data into meaningful information. It supports better decision-making, improves operational efficiency and helps Samsung Electronics manage its supply chain more effectively.

Notes:
The data used in this project is created for academic purposes only. It is not real confidential Samsung Electronics data.
